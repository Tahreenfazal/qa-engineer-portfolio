# QA Engineer Portfolio

This repository contains a QA engineering portfolio created to demonstrate practical knowledge of manual testing concepts.

## Skills Covered
- Manual Testing
- Test Case Design
- Test Scenario Writing
- Bug Reporting
- Test Planning
- SDLC & STLC understanding

## Project Scope
The project focuses on testing a sample web application login module to ensure functionality, usability, and reliability.

## Tools Used
- MS Excel / Google Sheets
- Markdown
- Git & GitHub

## Contents
- Test cases for login functionality
- Test scenarios
- Sample bug reports
- Test plan document

## Objective
To showcase hands-on QA skills relevant for entry-level QA Engineer roles.
